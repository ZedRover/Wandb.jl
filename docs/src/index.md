# Wandb.jl

Unofficial Julia Bindings for [wandb.ai](https://wandb.ai).

## Installation

For stable release:

```julia
] add Wandb
```

For the `main` branch:

```julia
] add Wandb#main
```

## Examples

To see the logging in action go [here](https://wandb.ai/avikpal/Wandb.jl). Detailed code for these examples can be accessed via the navigation menu.
